# MusicPlayer
Simple music player


## 操作方法
1. 安装后，首先要打开存储权限，不然直接打开会被告知“对不起，暂时没有音乐文件”;
2. 存储权限打开后，在手机文件目录下，排序->按修改时间排序->找到一个名为“playmusic”的文件夹，这就是我们要存歌曲的文件夹，或者我们可以手动创建playmusic文件夹;
3. 从手机下载几首音频文件，复制到步骤2所说的文件夹下，重新打开安装的应用，就可以使用了;
